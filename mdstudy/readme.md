### [Basic Syntax]

- Online editor : https://dillinger.io/
  - 1단계
    - 2단계
- # 제목1 - 최대 6개까지 제목
- ## 제목2
- ### 제목3
- #### 제목4
- ##### 제목5
- ###### 제목6
- 리스트는 웬만해선 (-)를 쓰자
- 애스터리스크(\*)를 앞뒤로 하나씩 쓰면 _이탤릭_
- 애스터리스크(\*)를 앞뒤로 두개씩 쓰면 **볼드**
- 세개는? **_볼드 + 이탤릭_**, 4개부터는 없음
- `작은 코드 블럭` : 앞뒤로 백틱(\`) 사용

> 인용구 : 맨앞에 > 사용

```
코드 블럭 : 앞뒤로 백틱(\`) 3개 사용
```
